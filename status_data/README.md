# 기술 설명

이 곳에서 설치에 관련된 이야기를 해주시면 좋습니다.

### aws-sdk

아래 사항들이 설치가 되어있어야합니다.

```
npm install aws-sdk
```

### axios

아래 사항들로 현 프로젝트에 관한 모듈들을 설치할 수 있습니다.

```
npm install axios
```

### terraform

```
terraform init
terraform plan
terraform apply
```
